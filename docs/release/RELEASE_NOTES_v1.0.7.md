# 🚀 Ginger Media v1.0.7 - The Gallery & Stability Update

Version 1.0.7 is a major milestone for media organization, featuring a completely redesigned Image Gallery and critical fixes for the playback experience.

## ✨ New Features

### 📂 Organized Image Gallery
- **Folder Sidebar**: Easily navigate through your photo collections with the new dedicated folder sidebar.
- **"All Photos" View**: A unified view to see every image in your library, regardless of which folder it's in.
- **Direct Management**: Add new image folders directly from within the gallery interface with the prominent "Add Folder" button.
- **Live Scanning Feedback**: Visual confirmation in the gallery header when the player is actively scanning for new content.

### 🖼️ UI/UX Enhancements
- **Refined Aesthetics**: Improved gallery layouts with better grouping and clearer typography.
- **Empty State Guidance**: New users are now greeted with a helpful onboarding view when their gallery is empty.

## 🛠️ Stability & Bug Fixes

### 🧵 Playback Robustness
- **Seek Bar Fixes**: Resolved the "disappearing seek bar" bug when tracks had unexpected metadata.
- **Smooth Clamping**: Improved seek bar interaction to prevent jumping or overflow on long tracks.
- **NaN Protection**: Hardened the progress tracking to handle edge cases in media duration reporting.

### ⚡ Performance
- **Optimized Scanning**: Dramatically reduced library scan times by skipping heavy metadata extraction for images.
- **UI Responsiveness**: Fixed potential hangs during large folder scans (Work in Progress for v1.0.7 final).
